Stats Glossary:
  • MIN = Minutes
  • PTS = Points
  • REB = Rebounds
  • AST = Assists
  • STL = Steals
  • TO = Turnovers
  • BLK = Blocks
  • Value = Projected Value (Projected Fantasy Points x 1,0000 / Salary)
  • AVG FPTS = Average Fantasy Points
  • Proj FPTS = Projected Fantasy Points
  • W/L = Win/Loss
  • FG-A = FGM-FGA
  • GP = Games Played
  • GS = Games Started
  • MPG = Minutes per Game
  • FG% = Field Goal %
  • RPG = Rebounds Per Game
  • APG = Assists Per Game
  • BLKPG = Blocks Per Game
  • STLPG = Steals Per Game
  • PPG = Points Per Game
  • FPPG = Fantasy Points Per Game
  • 3PT-A = 3PTM-3PTA
  • FPPM = Fantasy Points Per Minute
